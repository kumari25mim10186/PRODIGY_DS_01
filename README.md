# PRODIGY_DS_01
# 📊 Population Distribution Analysis

A Data Science project that visualizes the distribution of categorical and continuous variables using Python. This project demonstrates how to create a Bar Chart and Histogram to analyze demographic data such as Gender and Age distributions.

---

## 📌 Project Objective

The objective of this project is to analyze population data and visualize:

- Gender Distribution using a Bar Chart
- Age Distribution using a Histogram

These visualizations help in understanding patterns, trends, and distributions within a dataset.

---

## 🛠️ Technologies Used

- Python
- Pandas
- Matplotlib
- NumPy

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/your-username/Population-Distribution-Analysis.git
```

### Navigate to Project Folder

```bash
cd Population-Distribution-Analysis
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

```bash
python src/visualization.py
```

## 📈 Output

The project generates the following visualizations:

### Gender Distribution Bar Chart

- Displays count of Male and Female individuals.
- Useful for analyzing categorical data.

### Age Distribution Histogram

- Displays frequency distribution of ages.
- Useful for analyzing continuous numerical data.

Generated files:

```text

outputs/age_histogram.png
```
